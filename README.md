# Slastičarna Demo

Moderan web template za slastičarne izgrađen s Next.js, TypeScript, Tailwind CSS i Framer Motion.

## Značajke

- **Moderan dizajn** s animacijama i prijelazima
- **Responsive** dizajn za sve uređaje
- **Tamni način** podrške
- **Animirana navigacija** i komponente
- **Stranica za naručivanje** s prilagodbama
- **Galerija torti** s hover efektima
- **Sekcija recenzija** kupaca
- **Kontakt forma** i informacije
- **SEO optimiziran**

## Tehnologije

- Next.js 15
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React ikone
- Next Themes

## Pokretanje

1. Instalirajte dependencije:
```bash
npm install
```

2. Pokrenite development server:
```bash
npm run dev
```

3. Otvorite [http://localhost:3000](http://localhost:3000) u browseru.

## Struktura projekta

- `src/app/` - Next.js App Router stranice
- `src/components/` - Reusable komponente
- `public/` - Statični resursi

## Prilagođavanje

Template je dizajniran da bude lako prilagodljiv. Možete mijenjati:

- Boje u `tailwind.config.js`
- Sadržaj u komponentama
- Dodati nove stranice u `src/app/`
- Prilagoditi animacije u Framer Motion komponentama

## Deploy

Preporučuje se deploy na Vercel za najbolje performanse s Next.js.

```bash
npm run build
```
